# fdf
Infographie project map relief.

Installation : Launch:

Make

Usage : Execute program with path map file

./fdf maps/map_name.fdf

Control :

Translation

--> Y: Key: UP, DOWN
--> X: Key: LEFT, RIGHT
--> Z + : +
--> Z - : -

Zoom

--> zoom + : scroll up
--> zoom - : scroll down

Change View

--> ISOMETRIC : PageUp
--> PARALELLE : PageDown
--> Origin : O

Change Map

--> ENTER

Exit

--> ESC
